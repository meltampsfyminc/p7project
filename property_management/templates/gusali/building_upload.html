Expected Format

The GUSALI REPORT file should have:

  • Column 1: Building Code (A, B, C, D)
  • Column 2: Building Name
  • Column 3: Original Cost
  • Column 4: Classification
  • Column 5-9: Donation/Ownership Info
  • Column 10-15: Cost Changes
  • Column 16-19: Remarks/Totals

Data starts from Row 7. Row 6 contains headers.
